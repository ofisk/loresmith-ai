# Documentation Images

This directory contains screenshots and images used in the documentation.

## Images

- `welcome-screen.png` - Screenshot of the welcome screen shown when first accessing LoreSmith AI

## Adding New Images

1. Place image files in this directory
2. Reference them in markdown using: `![Alt text](images/filename.png)`
3. Use descriptive filenames
4. Optimize images for web (consider PNG for screenshots, optimize file size)
